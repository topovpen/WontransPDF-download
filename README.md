# WontransPDF — PDF 번역 뷰어 (다국어 양방향)

PDF를 불러와 **왼쪽=원본 / 오른쪽=번역본**으로 나란히 보여주는 Windows 데스크톱 앱입니다.
PDF 안의 **이미지·도표는 그대로 두고, 본문 텍스트만 번역해 같은 위치에** 다시 배치합니다.
번역본은 다시 PDF 파일로 저장할 수 있습니다.

> 이 저장소는 **프로그램 다운로드용**입니다. 최신 설치 파일은 아래 **[Releases](../../releases/latest)** 에서 받으세요.

---

## 📥 다운로드 & 설치

1. **[최신 릴리스](../../releases/latest)** 로 이동 → **Assets** → `WontransPDF-1.5-setup.exe` 다운로드
2. 내려받은 파일 실행
3. **SmartScreen 경고**가 뜨면 → **추가 정보 → 실행** (코드 서명 미적용이라 뜨는 것이며 기능엔 문제 없음)
4. 설치 마법사에 따라 진행 (관리자 권한 필요) → 시작 메뉴/바탕화면의 **WontransPDF** 로 실행

**시스템 요구사항**: Windows 10 / 11 (64비트). 별도 파이썬 설치 불필요.

**평가판 안내**: 최초 실행일로부터 **30일** 사용 가능한 평가판입니다. 창 제목에 남은 기간(`[평가판: N일 남음]`)이 표시됩니다. 자세한 조건은 설치 시 표시되는 사용권 계약(EULA)을 참고하세요.

## 🚀 빠른 시작

1. **파일 → PDF 열기**(Ctrl+O) → PDF 선택 → 번역할 페이지 범위 입력(기본: 전체)
2. 툴바에서 **엔진 / 언어(출발→목표) / 용도** 선택
3. (AI 엔진 사용 시) **도구 → API 키 설정**에서 키 입력 (아래 "API 키 설정" 참고)
4. **번역 실행(F5)** → 번역이 끝난 페이지부터 오른쪽에 표시됨
5. 필요하면 번역문을 클릭해 직접 수정
6. **파일 → PDF 저장**(Ctrl+S) 으로 번역본 저장

> 💡 이미 번역본이 있으면 번역 실행은 **선택한 페이지만 갱신(머지)** 하고 나머지는 그대로 둡니다.
> 전체를 다시 하려면 페이지 범위를 `all` 로 지정하세요.

## 🖥️ 화면 구성

상단에 **워드/엑셀식 메뉴바 + 슬림 툴바**가 있고, 아래에 원본(좌)·번역본(우)이 나란히 표시됩니다.

**메뉴바**
- **파일**: PDF 열기 · 최근 문서 · PDF 저장 · 캐시 관리 · 종료
- **번역**: 번역 실행(F5) · 선택 페이지 재번역(F6) · 현재 페이지 재번역(F7) · 번역 중지
- **편집**: 실행 취소/다시 실행 · 영역 편집 · 글자 크게/작게 · 원문 영역 지정 · 용어집
- **보기**: 확대/축소/맞춤(너비) · 처음/이전/다음/끝 페이지
- **도구**: API 키 설정 · 검색(Ctrl+F)
- **언어**: 한국어 / English (선택 시 재시작하여 UI 전체 전환)
- **도움말**: 정보

**툴바**: 자주 쓰는 선택 위젯(엔진·모델·언어·용도·줌·검색)과 실행 버튼(번역 실행·중지·저장), 원문 영역 지정 버튼이 있습니다.

## ✨ 주요 기능

### 번역 엔진 / 품질
- **다국어 양방향 번역**: 영어·독일어·일본어·중국어·한국어, 출발어 **자동 감지** 지원. 목표 언어별 전용 폰트 자동 적용.
- **엔진 선택**: `google`(무료, 키 불필요·기본값) / `deepl` / `claude` / `chatgpt` / `gemini`.
- **AI 모델 선택**: Claude(Haiku/Sonnet/Opus)·ChatGPT(4o-mini/4o)·Gemini(Flash/Pro). 모델별로 캐시가 분리되어 같은 문서를 모델만 바꿔 비교할 수 있습니다.
- **번역 용도(문맥) 지정**: 기술 문서·학술 논문·법률·의료 등 용도를 지정하면 그에 맞춰 자연스럽게 번역합니다.
- **DeepL 격식체/비격식체** 선택(지원 언어).

### 레이아웃 보존
- 원본/번역본 좌우 동시 보기(스크롤 동기화).
- **원본 줄 구조 보존**: 의도된 줄바꿈 단위로 분리해 같은 줄 구조로 번역.
- **소제목 자동 분리**: 굵게/글자 크기가 다른 머리글을 본문과 분리해 서식 유지.
- **2단/다단 레이아웃 인식**: 단 사이 통로를 문서 전체에서 일관 적용 → 본문이 옆 단을 침범하지 않음.
- **표 처리**: 셀을 가로 간격 기준으로 분리해 배치.
- 이미지·격자선은 그대로 두고 원문 글자만 제거 후 번역문을 얹습니다.

### 편집 / 수정
- **번역문 수동 수정**: 오른쪽 문단 클릭 → 텍스트·폰트·크기·**정렬(좌/가운데/우/양끝)**·굵게 조정.
- **원문 유지 선택**: 특정 문단을 번역하지 않고 원래대로 둘 수 있음. 엔진·모델별 **번역 이력**을 불러오거나 되돌리기 가능.
- **영역 편집**: 번역 문단 박스를 드래그로 이동/리사이즈.
- **현재 페이지 글자 크기 ±**: 해당 페이지 글자만 조절(재번역 없음).
- **반복 문구 일괄 적용**: 머리말·꼬리글처럼 전체 일치하는 문단을 전 페이지에 일괄 반영.

### 원문 영역(번역 예외 지정)
- **머리말/꼬리글 영역(전 페이지)**: 드래그한 띠를 **모든 페이지**에서 원문 유지 (예: 페이지 번호·URL이 깨지지 않음).
- **현재 페이지 원문 영역(F2)**: 드래그한 영역을 **이 페이지에서만** 원문 유지.

### 저장 / 캐시 / 기타
- **세션 캐시**: 번역·편집·영역·용어집 상태를 저장 → 같은 PDF를 다시 열면 **API 재호출 없이** 복원. **최근 문서** 메뉴로 빠르게 열기.
- **PDF 저장 시 폰트 서브셋**: 사용한 글자만 포함해 출력 용량을 크게 줄임.
- **API 키 암호화 저장**(Windows DPAPI, 현재 계정에 묶임).
- **전역 용어집**: 특정 단어/문장을 원문 유지 또는 지정 번역으로 강제(전체 일치).
- **검색**(Ctrl+F): 문서 전체 일치 개수 확인 → 첫 일치 이동 / 용어집 등록.
- **UI 언어 전환**: 한국어 / English.
- **안정성**: 한 페이지 실패해도 그 페이지만 원문 유지하고 계속 진행.

## 🔑 API 키 설정 (AI 엔진 사용 시)

`google` 엔진은 키 없이 무료로 쓸 수 있습니다. `deepl·claude·chatgpt·gemini`는 각 서비스의 API 키가 필요합니다.

- **도구 → API 키 설정** 에서 키 입력 (암호화되어 저장됨)
- 또는 환경변수로 지정: `DEEPL_API_KEY` / `ANTHROPIC_API_KEY` / `OPENAI_API_KEY` / `GEMINI_API_KEY`

> ⚠️ AI/DeepL 엔진을 쓰면 번역 대상 텍스트가 해당 서비스 제공자에게 전송됩니다. 민감한 문서는 주의해서 취급하세요.

# WontransPDF — PDF 번역 뷰어 (다운로드)

PDF를 **왼쪽=원본 / 오른쪽=번역본**으로 나란히 보여주는 Windows 데스크톱 앱입니다.
**이미지·도표는 그대로 두고, 본문 텍스트만 번역해 같은 위치에** 다시 배치합니다.

> 이 저장소는 **프로그램 다운로드용**입니다. 최신 설치 파일은 아래 **[Releases](../../releases/latest)** 에서 받으세요.

---

## 다운로드 & 설치

1. **[최신 릴리스](../../releases/latest)** 로 이동 → **Assets** → `WontransPDF-1.5-setup.exe` 다운로드
2. 내려받은 파일 실행
3. **SmartScreen 경고**가 뜨면 → **추가 정보 → 실행** (코드 서명 미적용, 기능엔 문제 없음)
4. 설치 마법사에 따라 진행 (관리자 권한 필요)

설치 후 시작 메뉴 또는 바탕화면의 **WontransPDF** 로 실행합니다.

## 주요 기능

- **다국어 양방향 번역**: 영어·독일어·일본어·중국어·한국어 (출발어 자동 감지)
- **번역 엔진 선택**: `google`(무료·키 불필요) / `deepl` / `claude` / `chatgpt` / `gemini`
  - AI 엔진은 **본인의 API 키**를 입력해 사용 (암호화 저장)
- **레이아웃 보존**: 이미지·표·다단 구조 유지, 본문만 번역해 같은 위치 배치
- **수동 교정**: 번역문 클릭 → 텍스트·폰트·크기·정렬 수정, 원문 유지 영역 지정
- **전역 용어집**: 특정 단어를 원문 유지 또는 지정 번역으로 강제
- **세션 캐시**: 같은 PDF를 다시 열면 API 재호출 없이 복원
